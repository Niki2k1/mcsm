The **Players** tab manages who can join your server and what they can do —
no commands required.

## Add operators and whitelist players

Look players up by username — their UUID and skin avatar are resolved from
Mojang automatically, so you always add the right account:

- **Operators** — grant or revoke op status.
- **Whitelist** — toggle whitelist mode and manage the allowed players.

Changes apply live on a running server (via RCON) and are kept in the server
configuration, so they survive restarts and edits.

## Moderate online players

When the server is online, the tab lists everyone currently playing, with
their avatar and name. From there you can:

- **Kick** a player off the server,
- **Ban** them (and unban later).
